# Инструкции по настройке поддомена на hostfly.by

## Пошаговая инструкция

### Шаг 1: Вход в панель управления
1. Перейдите на https://my.hostfly.by
2. Введите логин и пароль для входа в личный кабинет
3. Нажмите "Войти"

### Шаг 2: Переход в cPanel
1. В личном кабинете найдите вашу услугу хостинга
2. Нажмите кнопку "Войти в cPanel" рядом с услугой
3. Дождитесь загрузки панели управления

### Шаг 3: Создание поддомена
1. В главном меню cPanel найдите раздел "Домены"
2. Нажмите на подраздел "Поддомены" или "Subdomains"
3. Нажмите кнопку "Создать новый поддомен" (Create a Subdomain)

### Шаг 4: Настройка поддомена
1. **Домен**: Выберите ваш основной домен из выпадающего списка
2. **Поддомен**: Введите название поддомена (например, `delivery-api`)
3. **Папка документа**: Оставьте автоматически сгенерированную папку или создайте свою
4. Нажмите кнопку "Создать" (Create)

### Шаг 5: Проверка создания
После создания поддомена:
- Вы увидите сообщение об успешном создании
- Поддомен станет активным в течение нескольких минут
- URL будет выглядеть так: `https://delivery-api.ваш-домен.ru`

## Важные примечания

### Ограничения тарифов
⚠️ **Важно**: На тарифном плане "Cloud Мини" создание поддомена невозможно, так как данный тарифный план ограничен одним сайтом.

### DNS записи
Поддомен автоматически настроится с правильными DNS записями. Обычно это занимает от нескольких минут до 24 часов для полного распространения DNS.

### SSL сертификат
На хостинге hostfly.by SSL сертификаты обычно настраиваются автоматически. Если поддомен не доступен по HTTPS, подождите несколько часов или обратитесь в поддержку.

## Альтернативные способы создания поддомена

### Через расширенный DNS редактор
Если стандартный способ не работает:

1. В личном кабинете hostfly.by перейдите в раздел "Домены"
2. Найдите ваш домен и нажмите "Управление DNS"
3. Выберите "Расширенный DNS редактор"
4. Добавьте новую CNAME запись:
   - **Имя**: `delivery-api` (без точки и домена)
   - **Значение**: `ваш-основной-домен.ru`
   - **TTL**: 3600

### Через cPanel - Парковка доменов
1. В cPanel перейдите в раздел "Парковка доменов" (Parked Domains)
2. В поле "Новый паркованный домен" введите полный поддомен: `delivery-api.ваш-домен.ru`
3. Нажмите "Парковка домена"

## Проверка работы поддомена

После создания поддомена проверьте его работу:

1. **В браузере**: Перейдите на `http://delivery-api.ваш-домен.ru`
2. **Через консоль**: 
   ```bash
   nslookup delivery-api.ваш-домен.ru
   ```
3. **Онлайн инструменты**: Используйте сервисы проверки DNS

## Устранение проблем

### Поддомен не открывается
1. Подождите до 24 часов для распространения DNS
2. Очистите DNS кэш на вашем компьютере
3. Попробуйте с другого устройства/сети

### Ошибка 404
- Убедитесь, что в папке поддомена есть файлы
- Проверьте правильность настроек в cPanel

### Ошибка доступа
- Проверьте права доступа к файлам (должны быть 644 для файлов, 755 для папок)
- Убедитесь, что индексный файл (index.html или index.php) существует

## Следующие шаги

После успешного создания поддомена:

1. **Загрузите файлы API сервиса** в папку поддомена
2. **Настройте Node.js приложение** (если поддерживается хостингом)
3. **Протестируйте API endpoints**
4. **Интегрируйте с InSales**

## Поддержка

Если возникли проблемы:

2. Обратитесь в техническую поддержку hostfly.by
3. Используйте форумы сообщества хостинга
